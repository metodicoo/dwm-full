# Dwm Rice
The dependencies are only (I think, but I guess if you have errors you know how to fix):
- libXft
- libXinerama
- pamixer
- xsetroot

## Optional Dependencies
- pipewire
- feh
- lxrandr
- picom
- dbus

# Gentoo
(sudo or doas)
```bash
sudo emerge --ask x11-libs/libXft x11-libs/libXinerama
```
